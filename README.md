# 🚦 Traffic Accident Severity Prediction Using LightGBM and Real-Time Weather Data

# 📌 Overview

Traffic accidents are a major concern worldwide, leading to loss of life, injuries, and economic damage. This project leverages Machine Learning and real-time weather information to predict the severity of traffic accidents based on environmental, temporal, and road-related factors.

The system uses the LightGBM algorithm to analyze accident data and estimate accident severity, helping authorities and road users identify high-risk situations and improve road safety.

# 🎯 Objectives

Predict traffic accident severity using machine learning.

Analyze the impact of weather, time, and road infrastructure on accidents.

Integrate real-time weather data for practical predictions.

Identify accident-prone regions through hotspot analysis.

Visualize accident trends using spatio-temporal analysis.

Provide an interactive interface for users to assess accident risk.
